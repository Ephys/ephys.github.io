Hello there!

I'm Guylian Cox and I'm a 20-year-old currently studying Computer Science at Institut Paul-Lambin in Brussels. I'm mostly self-taught and, while classes are doing wonders at sharpening my algorithmic skills, my knowledge of web-related technologies comes primarily from online documentations such as the MDP (and Stack Overflow).

I've grown a passion for web development since I was 16, I loved from the beginning that this domain keeps evolving and keeping me interested.
Starting off with PHP5 for a few years along with the mandatory web languages that are HTML5, CSS3 and its preprocessors (LESS at first, followed by SCSS).

I then started learning Javascript and, with the advent of NodeJS, started getting more interested in the back-end than the front-end.

I tend to prefer stricter working environment, one of the reasons I also like Java and would like to explore C# and .NET, Typescript felt like a blessing when I discovered it.

my projects

Tinkers' Steelworks is a MinecraftForge mod I forked mid-2014 when its original author, known as toops, stopped working on it. You can find the original version here

At first I picked it up to port it to Minecraft 1.7.10 but it quickly became a playground with which I put the Java concepts I was studying at my school into practice. My work on that project however, was mainly focused on fixing bugs, improving existing features and creating an API to support the interactions with other game modifications.
On a personal level this project helped me improved my understanding of Java, along with the discovery of gradle and maven.
